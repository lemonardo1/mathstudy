# 📚 수학 완전정복 사이트

고등수학과 대학교 수학을 정복할 수 있는 종합 학습 사이트입니다.

## 주요 기능

### 1. 개념 학습
- **고등수학**: 수학 상, 수학 하, 수학 1, 수학 2, 미적분, 기하, 확률과 통계
- **대학교 수학**: 선형대수학, 다변수 미적분학, 미분방정식, 해석학
- 수학의 정석, 최상위 문제집 급의 상세한 개념 설명
- MathJax를 이용한 수학 기호 렌더링
- Plotly를 이용한 다양한 그래프 시각화

### 2. 문제 풀기
- 난이도별 문제 제공 (쉬움, 보통, 어려움)
- 과목별/주제별 필터링
- 단계별 해설 제공
- 정답 확인 기능

### 3. 질문하기
- 학습 중 궁금한 점 질문
- 질문과 답변 저장 기능

### 4. 복습 노트
- 중요한 개념 저장
- 복습 노트 관리 기능
- 로컬 스토리지를 이용한 데이터 저장

## 기술 스택

- **HTML5**: 구조
- **CSS3**: 스타일링 (반응형 디자인)
- **JavaScript (Vanilla)**: 기능 구현
- **Vite**: 개발 서버 및 빌드 도구
- **MathJax**: 수학 기호 렌더링
- **Plotly.js**: 그래프 시각화
- **Font Awesome**: 아이콘

## 설치 및 실행

### 사전 요구사항
- Node.js (v16 이상)
- npm 또는 yarn

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```
개발 서버가 실행되면 브라우저가 자동으로 열립니다. (기본 포트: 3000)

### 프로덕션 빌드
```bash
npm run build
```

### 빌드 미리보기
```bash
npm run preview
```

## 사용 방법

1. 개발 서버를 실행한 후 브라우저에서 사이트를 엽니다.
2. 왼쪽 사이드바에서 학습하고 싶은 과목을 선택합니다.
3. 챕터를 선택하여 개념을 학습합니다.
4. "문제 풀기" 탭에서 문제를 풀어보세요.
5. "질문하기" 탭에서 궁금한 점을 질문하세요.
6. "복습 노트" 탭에서 중요한 내용을 저장하세요.

> 참고: 개발 서버 없이 직접 `index.html` 파일을 브라우저에서 열어도 사용할 수 있습니다.

## 주요 특징

- **상세한 설명**: 각 개념을 자세히 설명하고 예제를 제공합니다.
- **다양한 그래프**: 함수, 수열, 확률분포 등을 시각화합니다.
- **반응형 디자인**: PC, 태블릿, 모바일에서 모두 사용 가능합니다.
- **로컬 저장**: 질문과 노트는 브라우저에 저장되어 유지됩니다.
